# Interview Review

该目录保存阶段性面试复盘，不存放未经验证的背诵答案。

每个主题建议包含：

1. 一句话概念定义。
2. 在 HSR 中的实际使用位置。
3. 端到端数据流或生命周期。
4. 一个真实 Bug 或工程取舍。
5. 可替代方案及其权衡。
6. 三道由浅入深的面试题。

优先主题：

- UObject、反射和 GC。
- Actor/Component 生命周期。
- UBT、UHT 和模块依赖。
- ASC Owner/Avatar、AttributeSet、Ability、Effect 和 Tags。
- GAS 与 TurnSystem 职责边界。
- DataAsset、DataTable、软引用和 Asset Manager。
- UMG 事件驱动更新。
- SaveGame Schema 与版本迁移。
